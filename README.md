<h2>⚙️ 主な機能</h2>
<p><ul>
<li>コース別有効判定: 距離やコース条件に応じた、各スキルの発動タイミングと有効性の自動判定。</li>

<li>加速期待値の可視化: 終盤開始地点からの発動ラグを計算し、加速としての価値を評価します。</li>

<li>カスタム設定: 終盤開始位置（LS）や最終直線位置などの手動入力による詳細なシミュレーション。</li>
</ul></p>
